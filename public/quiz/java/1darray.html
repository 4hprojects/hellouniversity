You will trace the execution of the following Java program. Complete the table by filling in the values for each iteration of the loop.

Program to Trace

int[] numbers = {4, 7, 10, 3, 6};
int total = 0;
int evenCount = 0;
double average = 0;

for (int i = 0; i < numbers.length; i++) {
    int current = numbers[i];
    int added = 0;

    if (current % 2 == 0) {
        evenCount++;
        added = current;
        total = total + current;
    }

    average = (double) total / (evenCount == 0 ? 1 : evenCount);

    System.out.println("i=" + i +
        " value=" + current +
        " added=" + added +
        " total=" + total +
        " evenCount=" + evenCount +
        " avg=" + average);
}
